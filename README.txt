Proyecto MALIBU - demo HTML listo para pasar a PHP.

Estructura:
- index.html: web principal
- nosotros.html: página institucional
- reservas.html: formulario de reservas
- menu/index.html: menú digital demo
- admin/index.html: panel administrativo demo
- assets/css/style.css: estilos generales
- assets/js/main.js: productos demo del menú
- assets/img: logo e imágenes recibidas

Cuando se pase a PHP:
- El formulario de reservas debe guardar en MySQL.
- El calendario debe leer días ocupados desde la base.
- El menú digital debe cargar productos/categorías desde tablas.
- El admin debe permitir CRUD de productos, categorías y reservas.
